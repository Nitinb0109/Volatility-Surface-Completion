# Implied Volatility Surface Reconstruction using XGBoost

## 📊 Overview
Reconstructed the Implied Volatility (IV) surface by predicting missing values in options data using XGBoost.

## ⚙️ Approach
- Feature engineering: moneyness, tau, interactions
- Model: XGBoost regression
- Data-driven IV prediction

## 📈 Results
- R² Score: 0.85

## 📂 Files
- solution.py → model code
- submission.csv → predictions
- methodology.pdf → explanation
- requirements.txt → dependencies
